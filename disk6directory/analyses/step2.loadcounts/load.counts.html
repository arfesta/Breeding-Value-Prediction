Alt Approaches Step 2: Load Raw Counts

Adam Festa

4/16/2018

Script objective

This script takes the raw bioloigcal replicate count data produced by the first script (analyses/step1.data_prep/README.md) and preps objects for normalization techniques. Given that the objective of this project is to predict phenotypes, we will subset the counts and phenotype data to only include families where we have phenotypes available to use.

  • Subset these families from txi object

    # Remove families with missing phenos from counts
txi$counts <- txi$counts[,-c(missing.phenos)]

# Filter out which transcripts have less than 10 total counts
rm.zero <- which(apply(txi$counts,1,sum) < 10)
txi$counts <- txi$counts[-c(rm.zero),]

# Remove low counts and remove fams with missing pheno from txi object
txi$abundance <- txi$abundance[-c(rm.zero),-c(missing.phenos)]
txi$length <- txi$length[-c(rm.zero),-c(missing.phenos)]
txi$variance <- txi$variance[-c(rm.zero),-c(missing.phenos)]
