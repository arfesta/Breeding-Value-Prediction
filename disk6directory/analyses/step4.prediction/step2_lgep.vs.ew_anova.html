Conduct ANOVA using transcripts & SNPs

  • First on transcripts
raw.txpt.pvals <- unlist(mclapply(1:ncol(train.bio.counts),function(each.txpt){
  anovaScores(x=train.bio.counts[,each.txpt],y = train.bio.phenos$Volume)
},mc.cores=8))
fdr.txpt.pvals <- p.adjust(raw.txpt.pvals,"fdr")
hist(fdr.txpt.pvals,main = "FDR adjusted Txpt p-vals LGEP")

  • Now on SNPs
raw.snp.pvals <- unlist(mclapply(1:ncol(train.bio.snps),function(each.snp){
  anovaScores(x=train.bio.snps[,each.snp],y = train.bio.phenos$Volume)
},mc.cores=8))
fdr.snp.pvals <- p.adjust(raw.snp.pvals,"fdr")
hist(fdr.snp.pvals,main = "FDR adjusted SNP p-vals LGEP")

Create test and train for prediction

  • Prediction will be done on family-mean values, so create test/train on family mean objects
